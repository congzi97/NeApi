HTTP 组件 - Request 类

Request 类 - Action列表

  • Action Name : getParams

    Action Explain : 获取URL的值!比如:url = http://xx.com/a/b/c;getParams[2]返回c

  • Action Name : get

    Action Explain : 获取$_GET数据

  • Action Name : post

    Action Explain : 获取$_POST数据

  • Action Name : getPostAll

    Action Explain : 获取全部$_POST数据

  • Action Name : getGetAll

    Action Explain : 获取全部$_GET数据

  • Action Name : isGet

    Action Explain : 判断是否ajax get访问

  • Action Name : isPost

    Action Explain : 判断是否ajax post访问

  • Action Name : isAjax

    Action Explain : 判断是否ajax访问

  • Action Name : http_request

    Action Explain : curl远程访问

使用示例

    
1、初始化对象 $obj = Ne::$app->request or $obj = Ne::$app->request($config);

    
2、$obj->action() ; action请按实际值更改
